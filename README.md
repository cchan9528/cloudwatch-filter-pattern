# cloudwatch-filter-pattern

Library to implement Cloudwatch Logs Filter Pattern Syntax https://docs.aws.amazon.com/AmazonCloudWatch/latest/logs/FilterAndPatternSyntax.html

# Usage

## Commands
- Build
    ```
    npm run release
    ```
- Test Only
    ```
    tsc && npx jest
    ```
- Format Only
    ```
    npm run format
    ```
